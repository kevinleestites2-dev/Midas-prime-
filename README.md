# 🥇 MidasPrime — The War Chest

*"Everything it touches turns to gold."*

MidasPrime is the **Economic Architect** of the Pantheon. It manages the capital, allocates resources, and ensures the self-sustainability of the 18-bot empire.

## 🏛️ Role in the Pantheon
MidasPrime is the **Treasury**. It doesn't just store wealth; it generates it. It works with **Open-trade** for market capital and **OmegaPrime** for labor-based income. It is responsible for the **War Chest**.

## 🚀 Key Features
- **Dynamic Allocation:** Automatically shifts capital between trading, working capital, and reserves based on performance.
- **Auto-Withdrawal:** Moves profits to the Forgemaster's personal wallet once thresholds are met.
- **Categorical Expansion:** Supports adding new "Earning Streams" to diversify the empire's income.

## 🛠️ Expanding the War Chest
The Forgemaster can add new ways to earn by running:
```bash
python3 midas_expand.py <new_category>
```
Example categories: `staking`, `saas_revenue`, `data_arb`, `content_mining`.
